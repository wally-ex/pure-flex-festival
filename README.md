# Pure Flex Festival 2026

Site oficial do festival de trap **Pure Flex**, que ocorrerá em abril de 2026.  
O objetivo do site é divulgar o evento, com informações sobre ingressos, countdown e artistas confirmados.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages (deploy)
- Fontes customizadas (unzialish, Bebas Neue, Inter)

---

## Estrutura do projeto

```
[pasta raiz]
│
├─ index.html          # Página principal
├─ css/                # Arquivos de estilo
├─ js/                 # Scripts do site
├─ fonts/              # Fontes customizadas
├─ images/             # Imagens usadas no site
├─ .gitignore          # Configurações do Git
└─ README.md           # Este arquivo
```

## Instruções de uso

### Para abrir localmente
1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/pure-flex-festival.git
```
2. Abra `index.html` no navegador ou use uma extensão como **Live Server**.

### Para atualizar no GitHub Pages
1. Faça alterações nos arquivos locais.
2. Commit e push das mudanças:
```bash
git add .
git commit -m "Mensagem do commit"
git push origin main
```
3. Aguarde alguns minutos e atualize a página no GitHub Pages.
